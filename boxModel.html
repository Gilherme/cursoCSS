Box Model

Box model ou modelo de caixa é como denominamos uma div, por exemplo que ocupa um espaço dentro da tela, esse espaço muitas vezes retangulares ou quadrados, é nomeado, Box Model

Tamanho dos Box

Podemos definir tamanhos das caixas com as propriedades

  • width: ; => Define a largura de um box os valores podem ser fixos como 300px ou 80%
  • height: ; => Definimos a altura de um box usando os mesmos parametros do width

Padding

Padding é essencial para alinharmos nossos conteudos dentro da caixa, o padding atua entre o conteudo dento da caixa até a borda da caixa, é esse espaçamento entre os dois, como uma margen interna

  • padding-top: ; => Espaçamento superior
  • padding-right: ; => Espaçamento a direita
  • padding-left: ; => Espaçamento a esquerda
  • padding-bottom: ; => Espaçamento inferior
  • padding: Valor1 ; => usando só a propriedade padding e passando um valor, ele por padrão adicionará o valor a todos os lados
  • padding: valor1 valor2; => Passando 2 valores, o primeiro sera aplicado padding no eixo y, ou seja, top e bottom e o segundo no eixo x, ou seja left e right

*IMPORTANTE Por padrão o padding se soma a ao tamanho do box, ou seja, se definir uma largura de 100px e adicionar um padding-left de 20px, logo o tamanho total da caixa passa a ser 120px

esse texto está dentro de uma caixa que inicialmente tinha 100px por 100px mas foram adicionadas paddings de 20px no eixo "y" e 30px no eixo "x" fazendo com que a caixa ficasse com o tamanho de 140px por 160px

Margin

Como o padding a margin serve para criar espaço entre coisas, mas no caso da mesma o espaçamento é feito entre caixas e não internamento, usando a propriedade margin: ; podemos dar valores como 10px, 15%, auto

  • margin-top: ; => Margem superior
  • margin-right: ; => Margem a direita
  • margin-left: ; => Margem a esquerda
  • margin-bottom: ; => Margem inferior
  • margin: Valor1 ; => usando só a propriedade margin e passando um valor, ele por padrão adicionará o valor a todos os lados
  • padding: valor1 valor2; => Passando 2 valores, o primeiro aplicará margin no eixo y, ou seja, top e bottom e o segundo no eixo x, ou seja left e right

*IMPORTANTE Quando se da uma margem a um box que está ao lado do outro as margens se fundem, por exemplo, a caixa 01 abaixo tem uma margim bottom de 10px e a caixa 2 tem uma margim top também de 10px, logo a margem entre os 2 é de 10px, e não de 20px. Caso a margin de um deles fosse diferente, e margem total entre os 2 seria a margem de maior, isso é chamado de Margin colapse
