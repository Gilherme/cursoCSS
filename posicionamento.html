Posicionamento

Para usar posicionamento usamos a propriedade Position: ; e podemos aplicar os valores abaixo. Por padrão todos os elementos assumem o position:static;

  • static
  • relative
  • absolute
  • fixed

Aplicar o Position: relative ; por si só não altera em nada o posicionamento do elemento, para isso precisamos usar outras propriedades, listada abaixo,

  • top: Ypx ; Onde Y quer dizer a quantidade de pixels a ser dada como "margin". As propriedades abaixo funcionam da mesma maneira e pode ser dado valores negativos também
  • right: Ypx ;
  • bottom: Ypx ;
  • left: Ypx ;

    • O alinhamento vai ser em relação a sua antiga posição

Quando aplicamos o valor absolute o elemento sai totalmente do fluxo natural e o que viria abaixo dele toma o lugar dele e sem aplicar as propriedades top, right, left e bottom, ele o sobrepoem

IMPORTANTE: O elemento sempe se baseara no container pai quando forem aplicada as propriedades de posição, porem é nesncessário que aplique no container pai o position: relative ;

O alinhamento vai ser em relação ao container pai

Usando o fixed o elemento fica parado e não acompanha o rolamento da pagina usano normalmente em menus suspensos ou chat no canto da pagina

Sobrepondo elementos

Quando aplicamos position: ; dependendo do valor, um alemento pode ficar sobre o outro

Com a propriedade z-index: x ; Onde x x é um numero, definimos qual elemento aparecerá na frente, o numero maior fica na frente. É importante que todos os elementos afetados estejam com o position: ;
