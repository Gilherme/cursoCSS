Elementos Flutuantes

Com a propriedade float: ; conseguimos fazer com que os box fiquem um ao lado do outro, possibilitando varias possibilidades, os valores disponiveis são:

  • right => O box flutuara a direita
  • left => O box flutuara a esquerda
  • none => O box não flutuará
  • right => O box flutuara a direita

*IMPORTANTE Quando dentro de um container aplicamos a propriedade float em alguns elementos, esses entram em um fluxo diferente do fluxo original da pagina, podendo gerar bugs, como por exemplo um elemento sobrepor outro, para isso não acontecer use a propriedade clear: ; e podemos dar os seguintes valores:

  • left => Limpa o fluxo flutuante a esquerda, funciona como se o conteudo pulasse para baixo
  • right => Limpa o fluxo a erquerda
  • both => Limpa o fluxo de ambos acima, de preferencia a esse
  • none => Não limpa, age como se não houvesse o clear

*IMPORTANTEQuando coloca elementos flutuantes dentro de uma div principal, os elementos flutuantes utrapassam as fronteiras da div principal, logo a div principal não se expande e é como se não exixtisse
