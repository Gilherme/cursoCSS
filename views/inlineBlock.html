Elementos em Bloco

Exixtem 3 tipos de elementos em css, que definem o comportamento dos blocos/caixas, são eles:

  • Block
  • inline
  • inline-block

Elementos Em Bloco - block

Elementos block ocupam toda a largura da pagina

Esse é um display block Esse é um display block Esse é um display block

Elementos Em Bloco - inline

Elementos inline ocupam apenas a largura do conteudo que ela contem

Esse é um box inline tags inline: span, a, img... Esse também é um box inline

Elementos Em Bloco - inline-block

Elementos inline-block ocupam apenas a largura do conteudo que ela contem e é possivel adicionar tamanhos

Esse é um box inline-block tags inline: span, div, table, p, h1, a, img... Esse também é um box inline-block

*IMPORTANTE Geralmente elementos inline como span não aceitam formatação de width and height, para mudar isso use inline-block
