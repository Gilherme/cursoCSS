Bacgrounds

Use bacgound-color: ; Para adicionar uma cor de fundo, podem ser cores rgba, exadecimal ou o nome por extenço, como, blue

Bacground-image

Use bacgound-image: url('enderecoDaImagem.jpg') ; Para adicionar uma imagem de fundo

Background-repeat

O padrão de quando se aplica uma imagem de fundo é que ela se repete até cobrir todo o espaço disponivel

Com a propriedade bacground-repeat: ; Podemos dar os valores:

  • no-repeat - A imagem não se repete
  • repeat ----- Repete a imagem nos eixos X e Y (Padrão)
  • repeat-x --- Repete a imagem no eixo X
  • repeat-y --- Repete a imagem no eixo Y

Bacground-attachment:;

Com a propriedade bacground-attachment: ; definimos se a imagem acompanhara o conteudo ou ficara estática. Podemos dar os seguintes valores:

  • fixed --- Faz a imagem ficar estática, parada no fundo, mesmo que o conteudo dá pagina desça ou suba
  • scroll --- A imagem acompanha o conteudo da pagina

Bacground-position

Com a propriedade background-position: eixo-x eixo-y ; podemos dar valores que definirão a posição da img de fundo Definomos a posição onde a imagem de fundo ficará. Com os seguintes valores:

  • left
  • center
  • right
  • top
  • bottom

É possivel acrescentar todos esses valores usando apenas uma propriedade, como no exemplo abaixo, usamos: bacground: gray url(img/yoshi.png) no-repeat scroll center ;

É importante que seja como na ordem acima bacground-color:; => bacground-image:; => bacground-repeat:; => bacground-attachment:; => bacground-position:;
