Formatando Links

Podemos mudar a cor de um link usando a propriedade color: cor ; Como já vimos em Textos

Para mudar estilos de links não visitados usamos a:link{ Estilos de texto }

Para mudar estilos de links visitados usamos a:visited{ Estilos de texto }

Para mudar estilos de links quando passa o cursor usamos a:hover{ Estilos de texto }

Para mudar estilos de ativos (quando clicamos) usamos a:active{ Estilos de texto }

IMPORTANTE Manter a ordem acima na hora de formatar os links
